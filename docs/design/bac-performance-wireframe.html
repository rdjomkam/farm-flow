  1. 1 Bac name — bac.nom, font-weight: 600, 13px
  2. 2 Rank badge — Sorted by FCR (best first). Background color-coded: accent-green-muted for #1, accent-blue-muted for mid, accent-red-muted for worst. Shows ranking reason.
  3. 3 Sparkline — 64x28px SVG polyline. All BIOMETRIE releve poidsMoyen values for this bac, chronologically. Stroke color matches rank: green=good, blue=ok, amber=warning.
  4. 4 Poids moyen — Latest biometrie poidsMoyen. accent-purple text.
  5. 5 Survie — vivants/nombreInitial. success text.
  6. 6 Biomasse — poidsMoyen x vivants / 1000. accent-blue text.
  7. 7 FCR — totalAlimentKg / gainBiomasseKg. accent-amber text. Turns accent-red if FCR > 2.0.
  8. 8 Last biometry date — Relative time ("il y a Xj"). muted-foreground text, 10px.
  9. 9 GMQ badge — Daily growth rate in g/day. Color-coded: green > 4 g/j, gray 2-4 g/j, red < 2 g/j. Key actionable metric.

6. Design Decisions Summary

Decision Choice Rationale
Tab toggle Croissance / Couts Keeps mobile cards compact. Farmers check growth daily, costs weekly.
Sort order By FCR ascending FCR is the most actionable metric — best performer first motivates.
Sparkline position Left of metrics (mobile), above metrics (desktop) Mobile: saves vertical space. Desktop: full-width sparkline is more readable.
GMQ badge Bottom-right with color coding Daily growth rate is the "glanceable" stat — green/gray/red instantly shows trend.
Ranking badge Top-right, subtle Creates healthy comparison without being aggressive. Gamification element.
6 metrics (growth tab) 3x2 grid Poids, Biomasse, FCR (row 1) + Survie, Vivants, Aliment (row 2) — prioritized.
Card interaction Tap → /bacs/{id} Consistent with existing VagueBacsSection behavior.
