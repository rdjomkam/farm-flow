# BUG-015 — Badge role ecrase le nom/email du membre sur mobile

**Severite :** Moyenne
**Detecte par :** @ronald
**Assigne a :** @tester (verification uniquement — fix deja applique)
**Sprint :** 9 (CR-009)
**Fichier(s) :** src/components/sites/site-detail-client.tsx

## Description

Sur la page detail d'un site, le badge du role (ex: "Administrateur", "Pisciculteur") est affiche sur la meme ligne que le nom et l'email du membre. Sur mobile (360px), le badge `shrink-0` compresse le nom et l'email qui sont tronques trop tot, rendant les informations illisibles.

## Etapes de reproduction

1. Ouvrir un site avec des membres
2. Observer la liste des membres sur mobile (360px)
3. Le nom et l'email sont tronques car le badge role prend trop de place horizontalement

## Cause racine

Le badge role est dans le meme conteneur `flex items-center` (ligne 246) que l'avatar, le bloc nom/email et le bouton actions. Le badge a `shrink-0` ce qui force le nom/email a se compresser.

## Fix

Deplacer le badge role sous l'email, dans le bloc `div.flex-1.min-w-0` (lignes 252-262), pour qu'il soit sur sa propre ligne.
