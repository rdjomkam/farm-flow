Build a single-page promotional website for DK Farm — a professional
catfish (Clarias gariepinus) farm in Cameroon. The site must convert
visitors into buyers and app subscribers. Language: French.

## Brand
- Name: DK Farm
- Tagline: "L'excellence piscicole au Cameroun"
- Tone: Professional, trustworthy, modern African agribusiness
- Colors: Deep greens + warm earth tones + clean whites (nature/freshness)
- NO purple gradients, NO neon, NO generic tech aesthetics

## Sections (in order)

### 1. Hero
Full-viewport. Bold headline about quality fish farming.
Primary CTA: "Commander maintenant" → WhatsApp link
Secondary CTA: "Découvrir nos produits" → scroll to products
Include a subtle animated background (water/waves motif).

### 2. Products (3 cards, bento grid)
- **Alevins (Fingerlings)**: Starting from 50 FCFA/unit. For farmers
  starting or restocking. Sizes: 3-5cm, 5-8cm, 8-12cm.
- **Poisson marchand (400g+)**: Ready-to-sell grown catfish.
  Price per kg. Fresh, farm-to-table quality.
- **Accès Farm Flow**: SaaS subscription for fish farm management.
  Track batches, mortality, feeding, growth. Mobile-first app.
  CTA: "Essai gratuit" → link to app registration.

Each card: product image placeholder, price range, key benefit, CTA button.

### 3. Why DK Farm (trust section)
3-4 proof points with icons:
- Qualité contrôlée (water quality monitoring)
- Livraison rapide (delivery across Cameroon)
- +X vagues réussies (track record)
- Application de suivi (technology-driven farming)

### 4. How to Order
Simple 3-step process:
1. Contactez-nous (WhatsApp/call)
2. Confirmez votre commande
3. Recevez vos poissons

### 5. Testimonials
2-3 farmer testimonials (placeholder names/quotes).
Cards with subtle entrance animation on scroll.

### 6. Contact / Footer
WhatsApp button (prominent, floating too), phone number,
location (Cameroon), social links placeholders.

## Technical
- Single HTML file with embedded CSS + vanilla JS
- Mobile-first (360px), responsive to desktop
- Fast loading, no external dependencies except Google Fonts
- WhatsApp link format: https://wa.me/237XXXXXXXXX
- Smooth scroll navigation
- Intersection Observer for scroll-triggered animations
